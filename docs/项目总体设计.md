# 项目总体设计

## 目标与边界

LiteTest-MAS 是面向测试生成任务的轻量多智能体系统。它比较 Text、Protocol、StateVector 和 SharedMemory 的质量与成本，同时将 private official tests 保持在 Agent 上下文之外。正式结论只适用于固定任务、固定模型和固定配置。

## 运行链

Planner、Memory、TestGen、Executor 与 Summarizer 构成逻辑协作链。Runner 从公开 manifest 读取任务，Candidate Parser 只处理模型候选，Private Sandbox 在隔离边界内运行 official tests，Evaluator 与 Strict Verifier 只公开脱敏计数和状态。

## 版本关系

M9 使用冻结的 Protocol V1、StateVector V1 和 FIFO SharedMemory V1；其有效 freeze 为 `cc7aac0417afb6acab47baaf7449459692fa9444`。M9.1 使用独立 V2 组件，其有效 freeze 为 `c79fd4826627bf61faf5d90540a014d243a59edd`。两者的公开聚合与结论不得混写。
