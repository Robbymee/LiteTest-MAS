# 演示操作指南

## 离线 Dashboard

从公开聚合构建离线页面：

```bash
python scripts/build_competition_dashboard.py --m9-dir reports/m9 --m9-1-dir reports/m9_1 --output-dir dashboard-output
```

在浏览器直接打开 `dashboard-output/index.html`。页面支持 M9/M9.1 切换，以及实验组、数据集、任务组和 Seed 视图；不会读取 private candidates、raw responses、hidden tests 或模型服务。

## 审计演示

```bash
python scripts/audit_competition_delivery.py --root . --allow-incomplete
```

命令输出 `valid=false` 时表示交付物仍有真实缺口，不是程序错误。严格模式不带 `--allow-incomplete` 会以退出码 1 表达同一结论。

## 边界

根目录的 `LiteTest-MAS赛事演示.pptx` 是十页可编辑中文演示文稿，`LiteTest-MAS演示视频.mp4` 是由同一公开内容生成的 1280×720 无声演示视频。二者可通过 `scripts/build_competition_presentation.py` 重建；构建依赖单独列于 `requirements-presentation.txt`，不影响 openEuler 核心运行依赖。

演示材料只使用公开聚合和脱敏结论，不包含参赛主体、private tests、candidate code 或 raw response。参赛承诺书仍需由真实参赛主体提供和签署。
