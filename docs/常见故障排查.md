# 常见故障排查

## 模块导入失败

从仓库根目录运行，并设置 `PYTHONPATH`：Windows 使用 `$env:PYTHONPATH = "$PWD;$env:PYTHONPATH"`；POSIX 使用 `export PYTHONPATH="$PWD:${PYTHONPATH:-}"`。随后执行 `python -m pytest tests -q` 或 `python3 -m pytest tests -q`。

## SSH 不可达

先检查 HostOnly 地址的 22 端口和 Hyper-V 虚拟机状态。不要为恢复连接而修改 openEuler DNS、软件源、证书或实验配置。网络恢复后先核对 `git rev-parse HEAD` 与工作区，再继续验收。

## 审计返回失败

严格审计返回 1 可能表示文档、交付物或安全缺口，并非脚本崩溃。使用 `--allow-incomplete` 生成报告，再按报告修复；不得删除报告或通过空白交付物消除缺口。

## Dashboard 构建失败

确认 M9 与 M9.1 报告目录含有效 manifest，且 Strict Verifier 为 `valid=true`。Dashboard 只接受公开聚合，不可替换为 private 运行目录。
